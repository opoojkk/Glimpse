# 更新记录

## v1.0.0 (2025-01-XX)

第一版，基本功能都有了：

**主要功能**
- 支持 ChatGPT 和 Claude
- 点击生成分享图片，有预览窗口
- Material Design 风格的代码块
- 自动识别平台，显示对应的 logo
- 自动提取对话标题
- 2 倍分辨率的高清图片
- 悬停显示分享按钮

**修复的问题**
- 修了 oklch 颜色格式导致的渲染失败（花了些时间）
- 修了行内代码的对齐问题
- 超长对话现在可以滚动了
- 移除了所有 class 避免样式冲突

**技术细节**
- 用的 Manifest V3
- marked.js 解析 Markdown
- html2canvas 生成图片
- 加了详细的调试日志
