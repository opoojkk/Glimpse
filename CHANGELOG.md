# 更新日志

## v1.0.0 (2025-01)

初始版本发布。

### 新增功能

- 支持 ChatGPT 和 Claude 平台
- 图片预览功能
- Material Design 风格代码块
- 平台自动识别与 logo 显示
- 对话标题自动提取
- 2x 高分辨率输出
- 悬停显示分享按钮

### 问题修复

- 修复 oklch 颜色格式渲染问题
- 修复行内代码对齐问题
- 优化超长对话显示（支持滚动）
- 移除 class 属性避免样式冲突

### 技术实现

- Manifest V3 规范
- marked.js Markdown 解析
- html2canvas 图片生成
- 详细调试日志

## 开发计划

1. **多种图片样式** - 多主题模板、自定义配色和布局
2. **多个对话** - 批量选择、编辑和重组对话内容
3. **多种图片格式** - PNG、JPEG、WebP、PDF 等格式支持
